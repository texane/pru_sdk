Programmable Real-time Unit (PRU) Software Support Package 
------------------------------------------------------------
============================================================
   LABS
============================================================

DESCRIPTION

   This directory includes source code for PRU labs, which provide a hands on 
   tutorial of the PRU, C-compiler, and Linux driver. The step-by-step lab 
   instructions are available at:

       http://processors.wiki.ti.com/index.php/PRU_Training:_Hands-on_Labs

   Lab solutions are provided in each lab directory.
       


WHAT LABS ARE INCLUDED?  

   The following labs are included in this directory. 

       LAB 1:  Toggle LED with PRU GPO
       LAB 2:  Read Push Button Switch on PRU0 GPI & Toggle LED with PRU1 GPO
       LAB 3:  Design a Temperature Monitor Application with the PRU
       LAB 4:  Introduction to Linux Driver
